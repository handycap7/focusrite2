Otros Modelos

Explora los diferentes modelos y encuentra el que mejor se ajuste a tu estudio.

Scarlett 2i2

Si necesitas más flexibilidad, el Scarlett 2i2 Studio te ofrece dos entradas combinadas para micrófono o instrumento, lo que lo hace perfecto para grabaciones en dúo o múltiples instrumentos.

Scarlett 4i4

Para la máxima conectividad, el Scarlett 4i4 Studio es la solución definitiva. Con cuatro entradas y cuatro salidas, esta interfaz es ideal para grabaciones más complejas o configuraciones avanzadas.

Scarlett Solo Studio

Incluye la interfaz Scarlett Solo, un micrófono de condensador de calidad de estudio, auriculares de precisión y todos los cables necesarios para que puedas comenzar a grabar de inmediato.
